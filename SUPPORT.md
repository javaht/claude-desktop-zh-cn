## 常见问题

### 安装相关

- **Q：安装后 Claude Desktop 没有切换为中文？**
  A：请确保完全退出 Claude Desktop 后重新启动。如果问题仍然存在，尝试重新安装。

- **Q：Windows 安装时提示权限不足？**
  A：右键以管理员身份运行安装程序。

- **Q：macOS 安装时提示安全警告？**
  A：在系统偏好设置中允许运行，或右键选择「打开」。

### 使用相关

- **Q：支持哪些 Claude Desktop 版本？**
  A：支持最新版本的 Claude Desktop。建议始终保持更新。

- **Q：是否支持第三方 API 模式？**
  A：本项目仅支持 API 使用方式，请确保已启用第三方模式。

01 特性 · Features

不只是把按钮换成中文。

装上就能用，出问题能退，新版本不丢字段。下面是它真正帮你处理的事。

01

一键安装，双平台通吃

macOS 双击 install-mac.command，Windows 右键以管理员身份运行 install-windows.bat，跟着菜单走完即可，不用手动改文件。

02

三种中文变体任选

简体中文 zh-CN、繁体中文·中国台湾 zh-TW、繁体中文·中国香港 zh-HK，安装时选一个语言即可。

03

安全备份 · 一键恢复

安装前自动备份原始 Claude，随时可从备份恢复或彻底卸载，装坏也退得回去。

04

智能合并，不丢字段

跟随你当前的 Claude 版本合并语言文件；新版尚未翻译的字段保留英文，绝不出现界面缺字。

05

第三方网关兼容

可关闭 3P gateway 对模型名的本地校验，deepseek、kimi 等自定义模型名不再让整套配置失效。

06

CC Switch skills 同步

把 ~/.cc-switch/skills 里的技能软链接进 Claude Desktop，同名跳过、源目录不动，可随时撤销。

登录后的 claude.ai 在线页面也汉化 菜单内一键开关自动更新 macOS 原生菜单同步汉化 安装写入失败自动容错

03 安装 · Install

两步装上，随时退得回去。

退出 Claude Desktop，跑一次入口脚本，按菜单选语言就行。两个平台分别看 ——

⌘
macOSinstall-mac.command
  1. 退出 Claude Desktop。
  2. 双击 install-mac.command，选择「安装中文补丁」。
  3. 选语言（1 简体 / 2 繁体·中国台湾 / 3 繁体·中国香港），按提示输入 Mac 登录密码。
  4. Claude 自动重启即生效；若没切换，到左下角账号菜单 Language 里手动选。

想更稳？ 菜单里还有「安全模式」，跳过 app.asar 结构性补丁，只做等长菜单汉化，兼容性更高。

⊞
Windowsinstall-windows.bat
  1. 退出 Claude Desktop。
  2. 右键 install-windows.bat，选择以管理员身份运行。
  3. 先选安装模式，再选语言（简体 / 繁·中国台湾 / 繁·中国香港）。
  4. 脚本自动备份资源、写入中文并重启 Claude。

三种模式： ① Cowork 兼容（跳过 app.asar）② 官方账号登录 ③ 第三方 API 实验。需要 Cowork 沙箱 / 截图工作区请选 ①。

↺ 所有改动只动本机 Claude 的本地资源，安装前自动备份。要恢复？再跑一次脚本，选「恢复 / 卸载」即可。
04 反馈 · Feedback

发现漏翻？一张截图就能修。

Claude 持续更新，新版偶尔会冒出没翻到的英文。看到了别忍着，提给我们就好。

提一个好 Issue

别只贴一句英文。带齐这四样，我们就能定位到资源层、稳稳修掉：

  • 01全屏截图 —— 保留导航、弹窗和上下文。
  • 02所在页面 —— 例如设置页、第三方供应商、工具详情。
  • 03触发操作 —— 点了哪个按钮，是否有报错。
  • 04期望译法 —— 有更顺的中文，直接写出来。

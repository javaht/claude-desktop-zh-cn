安装与反馈

一边保持安装简单，一边让漏翻反馈可执行。

使用者只需要下载并运行最新的 Rust / Tauri 安装器；反馈者只需要把截图和路径交代清楚。其余定位、补丁和验证由项目流程处理。

安装补丁

下载最新 Release 后，直接运行随版本发布的 Rust / Tauri 安装器，按提示完成安装或恢复即可。

  1. 1下载最新版本的安装包，按你的系统选择对应的安装器。
  2. 2运行安装器，按照界面提示完成中文补丁的安装、恢复或更新。
  3. 3完成后重新打开 Claude Desktop，确认最新资源已经生效。
→Tauri 安装器会统一处理安装流程，并保留与恢复相关的桌面端能力。
